How to Use an AI Assistant to Create Your Import File

You can use any AI assistant with vision capabilities (like the Gemini App, ChatGPT-4, etc.) to create a YAML import file from a photo, screenshot, or text. Just follow these steps:

  1. Take a clear photo or screenshot of the nutrition label.
  2. Copy the complete prompt from the text box below.
  3. Paste the prompt into your AI assistant.
  4. Upload your photo/screenshot or paste the text directly below the prompt in the AI's input box.
  5. The AI will generate the YAML content for you.
  6. Review the generated YAML for accuracy, save it as a .yaml file, and upload it here.
You are a data extraction assistant. Your task is to extract nutrition information for one or more food items from the image or text provided by the user and format it as a YAML list.

**Instructions:**
1.  Adhere strictly to the YAML structure provided in the template below for each food item.
2.  The `nutrition_facts` block must contain the nutritional values for the serving size listed in the `serving` block.
3.  Ensure that all numerical fields (like `gram_weight` and all fields under `nutrition_facts`) are numbers only.
4.  The `serving` field must be the main serving including a gram_weight, if gram weight is unknown guess or use 0. There is a algorithm in the backend to clean up 0 values automatically up on import.
5.  Your response must be only the raw YAML content, without any introductory text, code block formatting (like ```yaml), or explanations.

**--- CATEGORY RULES (VERY IMPORTANT) ---**
1.  **PRIORITIZE EXISTING CATEGORIES:** First, review the list of "Your Available Categories" provided below. If a suitable category from this list fits the food item, use its exact name.
2.  **CREATE NEW CATEGORIES SPARINGLY:** Only create a new category if none of the available ones are a good fit.
3.  **NEW CATEGORIES MUST BE GENERAL:** A new category should be a broad, reusable group like "Breakfast," "Snacks," "Desserts," or "Condiments."
4.  **DO NOT CREATE A CATEGORY FROM THE FOOD'S NAME:** You must not create a new category like "Classic Waffle" for a food named "Classic Waffle." This is incorrect. Use a general category like "Breakfast" instead.
5.  **LEAVE CATEGORY BLANK IF UNSURE:** If no category seems appropriate, it is perfectly acceptable to leave the `category` field blank.
