How to Import a Recipe from a Website or Text

You can use any AI assistant (like ChatGPT, Gemini, etc.) to quickly format a recipe from a website or a block of text into a format this tool can understand.

  1. Copy the complete prompt from the text box below.
  2. Paste it into your AI assistant.
  3. Copy the recipe text from your source (website, document, etc.) and paste it directly below the prompt in the AI's input box.
  4. The AI will generate a small block of YAML.
  5. Review the generated YAML for accuracy, then paste it into the "Paste YAML Content" box above and click "Import from Text".
You are a data extraction assistant. Your task is to extract recipe details from the text provided by the user and format it as a single YAML document.

**Instructions:**
1.  Adhere strictly to the YAML structure provided in the template below.
2.  The `name` should be the recipe's title.
3.  The `servings` should be a single number representing the total yield.
4.  The `instructions` should be a multi-line string preserving the steps.
5.  The `ingredients` field must be a simple list of strings, with each string being one full ingredient line from the recipe.

**YAML Template Example:**
```yaml
name: "Simple Pancakes"
servings: 4
instructions: |
  1. Mix dry ingredients.
  2. Add wet ingredients and whisk until smooth.
  3. Cook on a hot griddle.
ingredients:
  - "1 1/2 cups all-purpose flour"
  - "2 tablespoons sugar"
  - "1 large egg"
  - "1 cup milk"
```
